DOSBIAN is a Raspberry Pi distro created in 2020 by Carmelo Maiolino.

Keep in the palm of your hand an equivalent 486DX machine of the ninities and 
enjoy playing with your preferite retro software and retro games.

THE PROJECT
Web page: https://cmaiolino.wordpress.com/dosbian
Facebook: www.facebook.com/groups/Dosbian

Visit my web page to leave a comment, get the latest news, download new versions of Dosbian.
If you need help on how to start with Dosbian, please feel free to ask support to the Facebook community.

TERMS OF USE
Dosbian is a donationware project, this means you can modify, improve, customise it 
as you like, however, IT IS STRICTLY PROHIBITED TO USE IT FOR COMMERCIAL PURPOSES.

AWARDS
The existence of DOSBIAN couldn't be possible without the following gems: 

* Raspbian    - The Debian distro for Raspberry Pi (www.raspberrypi.org)
* DosBox      - The fantastic DOS emulator (www.dosbox.com) 
* LaunchBox   - A DOS games Frontend (www.launchbox-app.com)
* Ipxbox      - A Dosbox IPX-over UDP server (github.com/fragglet/ipxbox)
